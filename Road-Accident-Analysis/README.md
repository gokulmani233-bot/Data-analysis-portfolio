# Road Accident Analysis Dashboard

## Project Overview
This project analyzes road accident data to identify accident trends, casualty patterns, road conditions, and factors contributing to accidents. The dashboard helps stakeholders understand accident-prone areas and supports data-driven road safety decisions.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Objectives
- Analyze total accidents and casualties.
- Identify accident trends over time.
- Evaluate the impact of road types and road conditions.
- Understand vehicle involvement in accidents.
- Discover high-risk locations and accident patterns.

## Data Cleaning & Transformation
- Checked for missing values
- Removed duplicate records
- Standardized data formats
- Validated accident and casualty data
- Created calculated columns and measures for reporting

## Dashboard Features
- Total Accidents KPI
- Total Casualties KPI
- Monthly Accident Trend Analysis
- Road Type Analysis
- Road Surface Condition Analysis
- Vehicle Type Analysis
- Location-Based Accident Analysis
- Interactive Filters and Slicers

## Key Insights
- Identified accident-prone road conditions.
- Analyzed monthly accident patterns.
- Compared accidents across different road types.
- Evaluated casualty distribution and severity.
- Highlighted factors contributing to road accidents.
