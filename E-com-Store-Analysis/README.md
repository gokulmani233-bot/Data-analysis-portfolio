# E-Commerce Store Data Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data to understand customer behavior, sales performance, order trends, and product category performance. The dashboard provides insights that help businesses make data-driven decisions to improve sales and customer engagement.

## Tools Used
- Microsoft Excel
- Power BI
- Power Query
- DAX

## Dataset Information
The dataset contains:
- Order ID
- Customer ID
- Gender
- Age & Age Group
- Order Date
- Order Status
- Sales Channel
- Product Category
- Product Size
- Quantity Sold
- Sales Amount
- City & State Information

## Data Cleaning & Transformation
- Removed duplicate records
- Checked and handled missing values
- Standardized data formats
- Created Age Group classifications
- Validated sales and order information
- Prepared data for dashboard reporting

## Dashboard Features
- Total Sales KPI
- Total Orders KPI
- Category-wise Sales Analysis
- State-wise Sales Performance
- Gender-wise Purchase Analysis
- Age Group Analysis
- Monthly Sales Trends
- Order Status Distribution
- Channel-wise Sales Comparison
- Interactive Slicers and Filters

## Key Insights
- Identified top-performing product categories.
- Analyzed customer purchasing behavior by gender and age group.
- Compared sales performance across different states.
- Evaluated sales contribution from various sales channels.
- Tracked monthly sales trends and order patterns.
