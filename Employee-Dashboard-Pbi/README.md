# Employee Analytics Dashboard

## Project Overview
This project analyzes employee data to understand workforce demographics, salary distribution, departmental performance, and employee experience levels. The dashboard provides actionable insights to support HR and management decision-making.

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX

## Objectives
- Analyze employee demographics.
- Monitor salary distribution across departments.
- Evaluate employee experience levels.
- Track workforce composition and trends.
- Support data-driven HR decisions.

## Dataset Information
The dataset includes:
- Employee ID
- Employee Name
- Age
- Gender
- Department
- Job Role
- Experience
- Salary
- Performance Rating
- Location

## Data Cleaning & Transformation
- Handled missing values
- Removed duplicate records
- Standardized data formats
- Created calculated columns
- Built DAX measures for KPI reporting

## Dashboard Features
- Total Employees KPI
- Average Salary KPI
- Department-wise Employee Distribution
- Gender Distribution Analysis
- Experience Level Analysis
- Salary Distribution Analysis
- Performance Rating Analysis
- Interactive Filters and Slicers
- Drill-Down Visualizations

## Key Insights
- Identified departments with the highest employee count.
- Analyzed salary trends across different roles.
- Compared workforce distribution by gender and department.
- Evaluated employee experience levels.
- Highlighted performance trends within the organization.
